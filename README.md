# Book-Search-Engine

## Description and Usage
This app uses the Google Books API along with Apollo Server. This app was developed using the MERN stack. Users can search for books and will be and will be presented with options relavent to your search. Users can then login and save the books that they would like to view later.

## Tools Used

* JavaScript
* Node.js
* Express
* React
* Apollo Server
* Graphql
* Mongodb
* Mongoose
* Dotenv
* Heroku

## Installation
clone the repository to a local folder on your maching. In the command line `cd` to the root directory then run `npm i` to install all dependencies. To run the app on your local machine run `npm run develop` from the root directory. Then open `http://localhost:3000/` in your browser
